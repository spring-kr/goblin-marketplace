## 🎮 대화 모드 시스템

### 📱 4가지 대화 모드

1. **SINGLE** - 단일 대화
   - 즉석 질답
   - 컨텍스트 최소화
   - 빠른 응답

2. **CONTINUOUS** - 연속 대화
   - 이전 대화 기억
   - 점진적 발전
   - 자연스러운 흐름

3. **DEEP_DIVE** - 심화 탐구
   - 다각도 분석
   - 전문가 다수 참여
   - 종합적 응답

4. **CREATIVE** - 창의 협업
   - 아이디어 융합
   - 창의적 합성
   - 혁신적 관점
